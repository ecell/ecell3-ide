Introducing E-Cell IDE, a mighty biomodeling tool suite.

E-Cell IDE (Integrated Development Environment) is a GUI application that aims to support development of diverse biological models. IDE, the term originally coined by the software development industry, refers to a tool that provides comprehensive functions required in wide ranges of product development. E-Cell IDE provides a foundation for typical biological modeling processes on top of the state-of-the-art biological simulator, E-Cell System.

E-Cell IDE consists of three major components, pathway designer, simulator and analytical toolkit that are tightly integrated with each other.

Pathway Designer

Pathway designer is a visual modeling tool with which you can “draw” reaction networks in a very intuitive way. You can put reactants (Variables) and reactors (Processes) on a canvas and connect them with arrows or lines, just like creating a pathway diagram with your favorite drawing program. Thanks to the E-Cell System's semantics independent design, you can also build a non-biochemical model with Pathway Designer.

Simulator

Running simulation in E-Cell IDE is almost as easy as playing DVDs on a movie player. Once you get the species wired up, all you need to do is click the green triangle in the toolbar. Then, switch to viewer mode so that your pathway diagram will get animated “in sync” with the simulation.

Of cource you can always check out how the values are going through graph plotters and the property pane.

Analytical Toolkit

Basic analytical methods such as parameter estimation and sensitivity analysis are essential in comprehensive modeling processes. E-Cell IDE comes with five built-in analysis plug-ins; parameter estimation, robust analysis, sensitivity analysis and bifurcation analysis. You can always invoke the function from “Analysis” menu and start the analysis right within the IDE. In addition, you can write your own analyticalal plug-in in any programming language supported by CLR (Common Language Runtime), including C# and Python.

System Requirements

E-Cell IDE currently runs on .NET Framework 2.0 or later versions. Supported operating systems include: Microsoft® Windows® XP (32/64bit) and Windows Vista® (32/64bit).

Porting to other platforms using Mono are being planned.
