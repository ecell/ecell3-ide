E-Cell IDE - システムバイオロジーのためのモデリング・シミュレーション統合環境

　E-Cell IDE※は、Windows上で動作するシステムバイオロジー研究のための、GUIアプリケーションです。先進的で高度なシミュレーション技術を結集した、E-Cell3をシミュレーションエンジンとして組み込んだ、モデリング環境および総合的なシミュレーション・解析環境をユーザに提供する、ソフトウェアプラットフォームです。

　これは、シミュレーション研究者よる、モジュールの開発などのカスタマイズが可能な自由度の高い拡張性を持ち合わせつつ、効率の良いモデリングとシミュレーションを可能にする、実用的な統合環境であり、生化学・分子生物学の実験ラボの研究者のような、一般的な生物学者にとっても容易に使えるソフトウェアです。

　E-Cell IDEにはパスウェイレベルでのモデルエディター、シミュレーター、解析ツールキットが統合されており、一貫したルック・フィールでユーザに提供されています。

※　E-Cell IDEとは、E-Cell Integrated Development Environment （E-Cell 統合開発環境）の略称

パスウェイレベルでのモデル構築

　モデルの構築、操作のための方法として、E-Cell IDEでは複数の方法を用意しています。（１）プロジェクトエクスプローラー、（２）エンティティリストペイン、そして（３）ダイアグラムペインの３種類です。このいずれかの方法を用いてモデルの詳細を記述して構築することになります。

　プロパティペインでは、モデルにおけるエンティティオブジェクト（Variable，Process）に設定されたプロパティを表示し、編集することができる。

　ダイアグラムペインでは、モデルのネットワーク構造をパスウェイ・ダイアグラムとして描画し、画面上で各オブジェクトの位置関係を編集することができます。パスウェイ・ダイアグラムの配置については、一定のアルゴリズムに基づいて自動配置を行うオートレイアウト機能を提供しており、ユーザのモデリング作業を支援します。

シミュレーション実験

　E-Cell IDEにおけるシミュレーションの実行は簡単です。特別な設定、操作は必要なく、ただ、シミュレーションの開始、停止、一時停止をGUIにおけるボタン操作を行うだけでできます。DVDで映画を見るのと同様に容易に行うことが出来ます。

　シミュレーション実行中の、各物質量、活性の変化については、プロパティーペインによっていつでも参照することができ、摂動などの変化を与えることができます（E-Cell3による、オンタイム・シミュレーション実験）。

解析ツールキット

　システムバイオロジー研究でよく使われる解析ツールについて、E-Cell IDEにおいてシームレスに使用できる、解析ツール・ライブラリを提供しています。現在用意されている解析アルゴリズムは以下の４種です。ロバスト解析 (Robust Analysis)、感受性解析 (Sensibility Analysis)、分岐解析 (Bifircation Anaysis)、パラメータ推定 (Parameter Estimation)。この他、計算グリッドに対応しており、SGE(Sun Grid Engine)で構成されたバックエンドの計算クラスタに対してGlobus Toolkit経由で解析ジョブを送信して結果を得ることができます。

　これらの解析機能を含むE-Cell IDEの機能については、ユーザが自作したプラグインによるカスタマイズが可能です(Pythonスクリプトによる拡張、あるいは、C#によるプラグイン開発を行うためのSDKを提供)。

動作環境

　E-Cell IDE は、以下の環境において開発、動作検証を行っています。

  • .NET Framework 2.0 以降をインストールした、Windows® XP SP2以降(32/64ビット版両方をサポート)
  • Windows Vista® (32/64ビット版両方)
  • Windows 7® (32/64ビット版両方)

E-Cell IDEは、（独）科学技術振興機構(JST)の実施する、戦略的創造研究推進事業(CREST)、研究領域「シミュレーション技術の革新と実用化基盤の構築」、研究課題「システムバイオロジーのためのモデリング・シミュレーション環境の構築」（研究代表者 冨田勝 慶應義塾大学先端生命科学研究所所長）における研究成果の一部です。
