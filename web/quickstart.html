Tutorial: Quick Start

Tutorials on the E-Cell core system is available at the E-Cell project website.

Load & simulate a sample model

Load a sample model file

  1. From the menu, select:

    2. [File] -> [Import Model]

  2. Select "My Documents\My E-Cell Projects\sample\Drosophila.eml" from the list of files and click [OK].
    • The Drosophilla circadian rhythm model has now been loaded into the simulator.
    • The round objects represent Variables(usually correspond to substances), while the square objects represent Processes(reactions).

Run simulation

  1. Create Loggers
    • From the pathway, right-click [Pn].
    • Select [Create Logger] -> [Value].

      • You will see the flag [logged] on the right of the [Pn]. You need to set Logger for the Variables or Processes you want to trace during run time.

  2. Start simulation
    1. From the menu, select:

      2. [View] -> [Tracer]

      Objects with [logged] flag will be displayed in a Tracer Window.

    2. Input "100" in the text field on the right of the toolbar.
    3. From the listbox, select [sec].

      4. The simulation will run for 100 seconds by this setting.

    4. In the Tracer window, click [Save].

Observe effects of parameter variation

Here we change parameter [vd] in the reaction [toy_R10] from 0.95 to 2.0, and see how the model behaves.

  1. Click the Stop button from the toolbar.

    2. The simulation will be reset to time zero.

  2. Double-click the Process [R_toy10] from the EntityList on the left of the main window.

    PropertyEditor window will appear.

  3. Change value of [vd] to "2.0".
  4. Click [OK].
  5. Click Step button from the toolbar and run 100 seconds simulation.
  6. Click [Save] in the Tracer window.

Close the model

  1. From the menu, select: [File] -> [Close Project].
  2. Click [No] from the dialog to save the model.

Write a simple reaction model

Open a new project

  1. From the menu, select:

    2. [File] -> [New Project]

  2. Input "toy" in the text fields ([Name], [Comment], [Model Name]) of the NewProjectDialog.
  3. Click [OK].

Define Variables

  1. Create a new Variable:
    • Click the Variable button in the toolbar.
    • Click on the white area in the RootSystem A new Variable named "V0" is created.
  2. Create another Variable following the steps above.

Define a Process

  1. Create a new Process:
    • Click the Process button in the toolbar.
    • Click on the white area in the RootSystem A new Process named "P0" is created.
  2. Connect Variables with a Process.
    • Click the VariableReference button (arrow) in the toolbar.
    • Click on [V0].
    • Drag it over onto [P0].
    • Click on [P0] again.
    • Drag it over onto [V1].

      • Variable [V0] was defined as a substrate and [V1] was defined as a product of the reaction.

Change properties of the new Variables

Click the select button.

  1. Set the initial value of V0
    • Double-click [V0] in the pathway.
    • Enter 1000000 in the text field labeled [Value].
    • Variable [V0] now has the initial quantity of 1000000. .
    • Check the checkbox on the left of the [Value] field.

      • [V0] was also set as being logged during the simulatoin.

    • Click [OK].
  2. Set the initial value of V1
    • Double-click [V1] in the pathway.
    • Enter 0 in the text field labeled [Value].

      • Variable [V1] now has the initial quantity of 0.

    • Check the checkbox on the left of the [Value] field.

      • [V1] was also set as being logged during the simulatoin.

    • Click [OK].

Define properties of the Process P0

  1. Double-click [P0] in the pathway.
  2. For [classname], select "MassActionFluxProcess".
  3. Input "0.01" in the [k] text box.
  4. Click [OK].

Run simulation

  1. From the menu, select: [View] -> [Tracer].
  2. Set simulation time as "800" seconds.
  3. Click the Step button.

Exit E-Cell IDE

  1. From the menu, select:

    2. [File] -> [Exit].

  2. Click [OK] in the confirmation dialog to save the model .
  3. Click [Save] in the SaveProjectDialog.

    4. The model file and the saved simulation data will be placed in My Documents\My E-Cell Projects\toy.
